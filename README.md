<h1 align=center>MCFC</h1>
<p align=center>Minecraft Formatting Codes for Python 3.x </p>

> Using:
```py
from mcfc import f_print, colors

# example
f_print("&cSomething Red, &eSomething Yellow, &cR&eA&aI&bN&9B&dO&5W")

# list of available colors
colors()
```
